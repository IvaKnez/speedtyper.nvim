## üìã Installation

[packer](https://github.com/wbthomason/packer.nvim)

```lua
use {
    "speedtyper.nvim",
    branch = "main",
    config = function()
        require('speedtyper').setup({
            -- your config
        })
    end
}
```

[lazy](https://github.com/folke/lazy.nvim)

```lua
{
    "speedtyper.nvim",
    branch = "main",
    opts = {
        -- your config
    }
}
```
